# 자료구조: 배열(Array)
<br>

## 배열이란?
- 데이터를 나열하고, 각 데이터를 **인덱스**에 대응하도록 구성한 데이터 구조 
- 파이썬에서는 리스트 타입이 배열 기능을 제공하고 있음
<br>

## 배열이 왜 필요할까?
- 같은 종류의 데이터를 **효율적으로 관리**하기 위해 사용
- 같은 종류의 데이터를 **순차적으로 저장**
<br>

## 배열의 장단점
### 장점
- 빠른 접근 가능

### 단점
- 데이터 추가와 삭제에 비용이 많이 사용된다.
- 미리 최대 길이를 지정해야 함(java)

## 예제
- 1차원 배열
```python
data = [1, 2, 3, 4, 5]
print(data)
```
```python
[1, 2, 3, 4, 5]
```

- 2차원 배열
```python
data = [[1, 2, 3], [4, 5]]
print(data)
```
```python
[[1, 2, 3], [4, 5]]
```
